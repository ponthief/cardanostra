CardaNostra
Created by: Ponthief
Usage
1. Add Nostr Account - private key ('Nsec')
2. Add Nostr Relay(s) for Account to connect to
3. Add existing BOLTCARD:
- enter card's UID
- enter Nostr 'npub' of the user who owns the card
- enter card Name to use when issuing commands via Nostr
